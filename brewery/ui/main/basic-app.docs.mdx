This is a base-bone React application in Bit.

Fork the component to your workspace:

```bash
bit fork learnbit.apps/react/basic-app
```

Run the following to make this app loadable by your workspace:

```bash
bit use learnbit.apps/react/basic-app
```

Run the following to run this app:

```bash
bit run basic-app
```
